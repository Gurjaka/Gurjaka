### 🛠️ Projects & Contributions

I actively build tools and contribute to the Linux desktop ecosystem.

* **[Pretty](https://github.com/pretty-terminal/pretty)**
    An SDL3-based terminal emulator written in C. Designed to be portable, reliable, and aesthetically minimal.

* **[Qtile](https://github.com/qtile/qtile)**
    Contributor to the Wayland backend (`wayc`) and core C extensions. I focus on backend stability and functionality.

* **[Nixos-Dotfiles](https://github.com/Gurjaka/Dotfiles)**
    My personal system configuration. A fully reproducible NixOS setup using Flakes, heavily customized for a keyboard-driven workflow.

* **[DSGBase](https://github.com/dsg-org/DSGBase)**
    A hybrid C/Python framework for efficient dataset management, combining low-level backend logic with a Flask frontend.

* **[Nixpkgs](https://github.com/NixOS/nixpkgs)**
    Maintaining and updating packages within the Nix ecosystem.
